PV CS251 API Demo

หน้าเดโมคร่าว ๆ สำหรับโชว์ endpoint ที่มีอยู่ตอนนี้ และลองยิง API จากฝั่งฟร้อนได้เลย

ค่าเริ่มต้นตั้งเป็น backend local ที่ http://127.0.0.1:8000 และ endpoint ของ DRF router ใช้ trailing slash เช่น /api/catalog/products/

ถ้าเปิดไฟล์นี้ตรง ๆ ผ่าน file:// แล้วโดน CORS ให้เอาไฟล์นี้ไป serve ผ่าน frontend dev server ที่พอร์ต 3000 หรือวางเป็น template ฝั่ง Django แทน
